# my-sql-arena

Welcome to **my-sql-arena**, a dedicated sandbox and learning repository for mastering MySQL. This repository contains complete setup scripts, relational database schemas, table creation examples, and structured practice questions ranging from core fundamentals to advanced query optimization techniques.

## 🚀 Overview

This repository serves as a hands-on practice ground to understand how relational databases execute queries under the hood. It covers essential SQL paradigms including:
* **Core CRUD operations** and table relationships.
* **Query execution mechanics** (such as Row-by-Row execution in Correlated Subqueries vs. One-Time Evaluation in Non-Correlated Subqueries).
* **Data Aggregation** using `SUM()`, `COUNT()`, and `GROUP BY` optimization.
* **Dynamic text matching** and string manipulation via `LIKE` and `CONCAT()`.
